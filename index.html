Instrucciones rápidas para subir a GitHub y publicar con GitHub Pages:

  1. Inicia un repositorio nuevo en GitHub (público).
  2. Clona el repo y añade este archivo como index.html. Ejemplo de comandos: 
    git init
git add index.html
git commit -m "Add perfil persona website"
git branch -M main
git remote add origin https://github.com/tu-usuario/tu-repo.git
git push -u origin main
  3. En la configuración del repo, habilita GitHub Pages desde la rama main y la carpeta / (root). El sitio quedará publicado en https://tu-usuario.github.io/tu-repo/ en unos minutos.

Consejo: sube también el PDF (por ejemplo a la carpeta /assets) y cambia el atributo href del botón "Descargar mi perfil (PDF)" por el enlace ./assets/Perfil_Persona.pdf o por el enlace raw de GitHub.
